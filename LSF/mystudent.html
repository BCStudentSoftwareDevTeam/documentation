My Student Interface Testing Scenarios

Download Button 

Testing Steps:
  1. Click on Download button
  2. A modal pops up prompting user to select the students whose labor history they want to download
  3. Check the Select All checkbox ( * )
  4. Click Download
  5. Uncheck the Select All checkbox
  6. Check Alex Bryant checkbox ( ** )
  7. Click Download
Expected Outcome:

( * ) On click of download a csv file is downloaded that contains labor history of all students.  

( ** ) On click of download a csv file is downloaded that contains labor history of Alex Bryant. 

Verification:

labor history of student(s) is downloaded as a csv file
