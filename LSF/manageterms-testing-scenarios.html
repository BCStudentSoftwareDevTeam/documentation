Manage Terms Interface Testing Scenarios

Open a term / Open multiple terms

Testing Steps: [1]
  1. Go to Administration tab and click on Manage Terms
  2. Open Academic Year 2020-2021
  3. Set the Start Date to 01/01/2020 ( * )
  4. Set the End Date to   05/01/2020  ( * )
  5. Set the Primary Due Date to  03/01/2020 ( * )
  6. Set the Adjustment Due Date to 04/30/2020 ( * )
  7. Click on Close button, located at the end of the table. The button color changes to green and the button text changes to Open. ( ** ) 

NOTE: [1] Follow the same steps for opening multiple terms. 

Expected Outcome:

( * ) A blue flash appears at the top of the page informing user that the changes they have made is saved. 

"The {DATE FIELD} for AY 2020-2021 was changed to {DATE}"
 

( ** ) A green flash appears at the top of the page with the following notification:  

"The state for AY 2020-2021 is set to 'Open'."

Verification:
  1. Click on New Labor Status Form located in the sidebar
  2. Select Term select picker
  3. The term you just opened is in the term drop down menu
